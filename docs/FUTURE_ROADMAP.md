# Future Roadmap for £Per

This document outlines planned enhancements and future features for the £Per due-diligence engine. These modules are designed to expand the platform's utility and market reach.

## 👤 User Accounts and Personalization
*   **Saved Properties:** Allow users to save properties of interest, including all collected data points, for later review. This requires a user authentication system and a persistent storage mechanism (e.g., Cloudflare D1 or KV).
*   **Notes/Annotations:** Implement a rich text editor allowing users to add private notes and annotations to specific properties or data panels.
*   **Alerts:** Set up automated alerts (e.g., via email or in-app notification) when a property in a saved portfolio meets specific criteria (e.g., price drops below a threshold, or a new sale occurs in the area).

## 💰 Financial & Investment Tools
*   **Portfolio Mode:** A dedicated view for managing multiple properties simultaneously. This would allow for multi-property comparison charts and aggregated portfolio metrics (e.g., total potential value, average yield).
*   **Yield Calculation:** Integrate tools to calculate potential rental yield based on sale price and local rental market data.
*   **Tax/Cost Analysis:** Add modules to estimate potential purchase costs, stamp duty, and other associated fees.

## 📈 Additional Data Panels
*   **Crime Data:** Integrate data sources for local crime statistics (e.g., police data APIs) to provide a comprehensive risk assessment.
*   **Planning Applications:** A module to check the local council's planning application database for potential future developments or restrictions on a property.
*   **Energy Tariffs:** Integrate data on local energy tariffs and potential savings/costs.
*   **Utility Infrastructure:** Expand utility data beyond water to include gas, electricity grid capacity, and fiber optic coverage details.

## ⚙️ System Enhancements
*   **Multi-Property Comparison:** Enhance the current comparison view to handle more than two properties side-by-side, making it a core feature.
*   **Advanced Filtering:** Implement advanced filtering and sorting options for saved properties and search results.
*   **Customizable Dashboards:** Allow users to select which data panels are visible on the main dashboard, tailoring the view to their specific investment focus.
