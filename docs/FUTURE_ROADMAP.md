# £Per Future Roadmap

This document outlines the planned future features and enhancements for the £Per engine. These features are designed to expand the platform's utility, increase data depth, and improve user experience.

## 🚀 Phase 2: Feature Expansion (Next 6-12 Months)

### 1. User Accounts and Authentication
*   **Goal:** Allow users to save searches, track properties, and manage portfolios.
*   **Implementation:** Integrate a user authentication system (e.g., OAuth, email/password).
*   **Impact:** Shifts the platform from a single-use tool to a personalized service.

### 2. Portfolio Management Mode
*   **Goal:** Allow users to input multiple properties (via UPRN list or manual entry) and view a comparative dashboard.
*   **Implementation:** Develop a multi-property view that aggregates data panels side-by-side, highlighting variances (e.g., comparing EPC ratings across a portfolio).
*   **Impact:** Transforms the tool into a professional investment analysis platform.

### 3. Advertising and Monetization
*   **Goal:** Introduce non-intrusive, relevant advertising slots.
*   **Implementation:** Reserve space on the dashboard for premium services (e.g., mortgage brokers, local estate agents).
*   **Impact:** Provides a revenue stream to support the high cost of data acquisition.

## ✨ Phase 3: Data Depth and Integration (12+ Months)

### 1. Utility and Infrastructure Deep Dive
*   **Goal:** Integrate more granular utility data.
*   **Implementation:** Add modules for local council tax bands, specific utility connection details, and local planning permissions.
*   **Impact:** Increases the depth of due diligence, moving beyond basic infrastructure checks.

### 2. Historical Trend Analysis
*   **Goal:** Visualize property value trends over time.
*   **Implementation:** Integrate historical sales data (beyond just the last few years) and overlay it with macro-economic indicators (e.g., interest rate changes).
*   **Impact:** Provides predictive value, making the tool more valuable to investors.

### 3. Advanced Geospatial Analysis
*   **Goal:** Incorporate advanced spatial analysis.
*   **Implementation:** Calculate metrics like "walkability score" based on proximity to multiple transport links, or "school catchment area score."
*   **Impact:** Moves the tool from data aggregation to true predictive analysis.
